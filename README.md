# Event Seating Map

An interactive seating map application built with React, TypeScript, and Tailwind CSS.

## Features

- Interactive SVG-based seating map
- Seat selection with persistent storage
- Accessibility compliant (keyboard navigation, ARIA labels)
- Responsive design for desktop and mobile
- Performance optimized for large venues (15,000+ seats)
- Zoom and pan controls

## Optional Stretch Goals Achieved

- Heat-map toggle that colours seats by price tier.
- Dark-mode toggle that meets WCAG 2.1 AA contrast ratios.

## Libraries Used

- Lucide react for icons
- React-router-dom for routing
- scroll-behavior-polyfill for smooth scroll

## Running the Application

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```
